# mac-commondX
mac上实现文件文件夹的剪切移动

## 功能
- 在 Finder 中使用 `Cmd+X` 剪切文件/文件夹
- 使用 `Cmd+V` 移动到目标位置
- 支持原生撤销 (Cmd+Z)
- 3天免费试用，之后需激活
